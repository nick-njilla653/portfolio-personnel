## 📘 Introduction

Le **Modèle IA Mbong** est un projet personnel développé en **Décembre 2024** visant à créer un système d'intelligence artificielle pour la proposition de repas locaux camerounais. Ce projet combine l'expertise en Machine Learning avec la culture culinaire camerounaise pour offrir des recommandations personnalisées basées sur les exigences caloriques et les ingrédients disponibles.

---

## 🎯 Objectifs du Projet

- **Recommandation culinaire** : Proposer des repas équilibrés selon les préférences utilisateur
- **Personnalisation** : Adapter les suggestions aux exigences caloriques individuelles
- **Culture locale** : Intégrer la richesse de la cuisine camerounaise
- **Interface intuitive** : Créer une expérience utilisateur fluide et accessible

## 🤖 Algorithmes IA Implémentés

### 1. Filtrage Collaboratif
```python
def collaborative_filtering(user_id):
    # Trouver des utilisateurs similaires
    similar_users = find_similar_users(user_id)
    
    # Recommander des recettes populaires parmi les utilisateurs similaires
    recommendations = get_popular_recipes(similar_users)
    
    return recommendations
```

### 2. Filtrage Basé sur le Contenu
```python
def content_based_filtering(user_preferences):
    # Analyser les préférences utilisateur
    user_profile = create_user_profile(user_preferences)
    
    # Trouver des recettes similaires
    similar_recipes = find_similar_recipes(user_profile)
    
    return similar_recipes
```

### 3. Système Hybride
```python
def hybrid_recommendation(user_id, user_preferences):
    # Combiner les deux approches
    collaborative = collaborative_filtering(user_id)
    content_based = content_based_filtering(user_preferences)
    
    # Fusionner et pondérer les résultats
    final_recommendations = merge_recommendations(
        collaborative, content_based
    )
    
    return final_recommendations
```
