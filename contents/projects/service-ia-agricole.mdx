## 📘 Introduction

Le **Service IA pour Plateforme IoT Agricole** est un projet académique développé à l'École Nationale Supérieure Polytechnique de Yaoundé (ENSPY) en **Décembre 2024**. Ce projet vise à intégrer l'intelligence artificielle dans une plateforme IoT agricole pour optimiser les processus de culture et améliorer les rendements.

---

## 🎯 Objectifs du Projet

- **Analyse des sentiments** : Comprendre les retours utilisateurs sur la plateforme
- **Prédiction des cultures** : Recommander les meilleures cultures selon les conditions
- **Prédiction des rendements** : Estimer les productions agricoles
- **Explicabilité des modèles** : Rendre les prédictions IA compréhensibles
- **Système de recommandation** : Suggestions personnalisées pour les agriculteurs

## 📊 Modèles IA Implémentés

### 1. Analyse des Sentiments
```python
from transformers import pipeline

class SentimentAnalyzer:
    def __init__(self):
        self.classifier = pipeline(
            "sentiment-analysis",
            model="nlptown/bert-base-multilingual-uncased-sentiment"
        )
    
    def analyze_sentiment(self, text):
        result = self.classifier(text)
        return {
            'sentiment': result[0]['label'],
            'confidence': result[0]['score']
        }
```

### 2. Prédiction des Cultures
```python
from sklearn.ensemble import RandomForestClassifier

class CropPredictor:
    def __init__(self):
        self.model = RandomForestClassifier(
            n_estimators=100,
            random_state=42
        )
    
    def predict_crop(self, soil_data, climate_data):
        features = self.preprocess_features(soil_data, climate_data)
        prediction = self.model.predict(features)
        probability = self.model.predict_proba(features)
        
        return {
            'crop': prediction[0],
            'confidence': max(probability[0])
        }
```

### 3. Prédiction des Rendements
```python
import tensorflow as tf

class YieldPredictor:
    def __init__(self):
        self.model = tf.keras.Sequential([
            tf.keras.layers.Dense(64, activation='relu'),
            tf.keras.layers.Dropout(0.2),
            tf.keras.layers.Dense(32, activation='relu'),
            tf.keras.layers.Dense(1)
        ])
    
    def predict_yield(self, environmental_data):
        prediction = self.model.predict(environmental_data)
        return prediction[0][0]
```

### 4. Explicabilité avec SHAP
```python
import shap

class ModelExplainer:
    def __init__(self, model, training_data):
        self.explainer = shap.TreeExplainer(model)
        self.training_data = training_data
    
    def explain_prediction(self, input_data):
        shap_values = self.explainer.shap_values(input_data)
        
        return {
            'shap_values': shap_values,
            'feature_importance': self.get_feature_importance(shap_values)
        }
```
